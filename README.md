# BNV

### Summary
BNV responsive website.

### Local Installation


```bash
# 1. Clone Repo
git clone https://github.com/KyleCornelius/KyleCornelius.github.io.git

# 2. install dependencies
npm install

# 3. Use gulpfile to run application in default browser and see live changes
npm start
```
